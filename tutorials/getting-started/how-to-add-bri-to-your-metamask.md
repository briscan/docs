# How to add BRI to your Metamask

1. To set up BRI as a new network on Metamask. Click on the network selector at the top of the app and then choose "Custom RPC" from the list:   ![](../../.gitbook/assets/etz1%20%281%29.png)  
2. Then in the "New RPC URL" enter this address: New rpc URL: **`https://mainnet-rpc.briscan.io`**

{% hint style="success" %}
Optionally you can add the full parameters:

**Additional**: [https://mainnet-rpc.briscan.io](https://mainnet-rpc.briscan.io)  
**ChainId**: 0x7a  
**Explorer**: [https://briscan.io](https://briscan.io/)  
**Symbol**: BRI
{% endhint %}

![](../../.gitbook/assets/ez2.png)  


{% hint style="danger" %}
**Note: Please make sure you send BRI assets to only other bri chain wallets and not any exchanges. We won't be able to retrieve the funds in case bri assets are sent to non BRI chain supported wallets/exchanges.**
{% endhint %}
